## 1. Supabase schema

1. Open [Supabase Dashboard](https://supabase.com/dashboard) → NFOP project → **SQL Editor**.
2. Run `supabase/schema-calendar.sql` (after `schema.sql` / `projects` exists).
3. Confirm **Table Editor** → `supervisor_blocks`.
4. **Database → Replication** → `supabase_realtime` includes `supervisor_blocks`.

## 2. Local config

1. Copy `config.local.example.js` → `config.local.js`.
2. Set `NF_CONFIG_LOCAL.supabase.url` and `anonKey` (never commit `config.local.js`).

## 3. Live Server (two machines)

1. Open the **NFOP-4.1** folder in VS Code / Cursor.
2. **Live Server** → serve folder root (e.g. `http://127.0.0.1:5500` or LAN IP).
3. Repeat on the second machine (same Supabase project + `config.local.js`).
4. Both browsers must use **http/https** (not `file://`) so Supabase + Realtime work.
